# Fizi - Strona Wsparcia

Strona wsparcia dla aplikacji Fizi, opublikowana na GitHub Pages.

## Publikacja

Strona jest dostępna pod adresem: `https://fizzyappkm-bit.github.io/support_fizi/`

## Lokalne uruchomienie

Możesz otworzyć plik `index.html` bezpośrednio w przeglądarce lub użyć lokalnego serwera:

```bash
python3 -m http.server 8000
```

Następnie otwórz w przeglądarce: `http://localhost:8000`
